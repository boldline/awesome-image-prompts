## 💡 通用技巧

1. **风格融合**: 尝试 `watercolor meets digital art` 等跨界混搭
2. **色彩体系**: `complementary colors`（互补色）、`monochromatic`（单色系）、`split-complementary`
3. **版式构图**: `centered composition`、`asymmetric layout`、`golden ratio`
4. **质感层次**: `textured paper`、`canvas grain`、`smooth gradient`
