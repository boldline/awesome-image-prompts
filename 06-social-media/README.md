## 💡 通用技巧

1. **尺寸规范**:
   - Instagram: 1080×1080 (方图), 1080×1350 (竖图), 1080×1920 (Story)
   - 小红书: 1080×1440 (3:4竖图)
   - YouTube 缩略图: 1280×720
   - Twitter: 1200×675
2. **文字安全区**: 重要元素避开上下各 15% 区域（平台UI遮挡）
3. **色彩吸引力**: 高饱和 + 对比色更容易在信息流中脱颖而出
4. **情绪传达**: 面部表情、动态姿势比静态物体更有吸引力
