## 💡 通用技巧

1. **背景选择**: `pure white background`（白底标准图）、`gradient background`（渐变）、`contextual environment`（场景融入）
2. **光线标准**: `soft diffused studio lighting`（柔光）最通用，`rim light`（轮廓光）增加质感
3. **角度规范**: `45-degree angle`（主图标准角度）、`top-down flat lay`（俯拍）、`eye-level straight-on`（正面平视）
4. **材质表现**: `glossy reflection`、`matte texture`、`translucent`、`metallic sheen`
