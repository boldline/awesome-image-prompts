## 💡 通用技巧

1. **光线控制**: 使用 `Rembrandt lighting`（伦勃朗光）营造立体感，`butterfly lighting`（蝶形光）适合美妆
2. **视角关键词**: `eye-level`（平视）、`slightly above`（微俯）、`three-quarter view`（3/4侧面）
3. **肤质描述**: `porcelain skin`（瓷肌）、`freckled`（雀斑）、`sun-kissed`（阳光感）
4. **情绪表达**: `confident smile`、`pensive gaze`、`joyful laughter`
