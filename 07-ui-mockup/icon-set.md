## Prompt 1: 线性图标系统

> 通用型线性图标集，适用于 App 和 Web 项目

**Prompt:**

```text
A cohesive line icon set of 20 icons for a productivity SaaS application, displayed in a 5x4 grid on clean white background with subtle grid lines visible, consistent style: 2px stroke weight, rounded line caps and joins, designed on 24x24px grid with 2px padding, monochrome dark gray (#374151) color, icons include: home, search, inbox, calendar, clock, user, team, folder, document, chart-bar, settings-gear, notification-bell, camera, link, download, upload, trash, edit-pencil, star-favorite, checkmark-circle, each icon with clear label text below in small gray caption, optical size consistency (all icons feel the same visual weight), pixel-perfect alignment on grid, Lucide/Feather icons quality reference
```

**示例效果：**

![线性图标系统](images/icon-set-01.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 尺寸 | 1024×1024 | 正方形展示 |
| 风格 | Line Icons | 线性图标 |
| 模型 | GPT-Image-2 | 推荐使用 |

**变体建议：**

- 将笔触从 2px 改为 1.5px 获得更精致的感觉
- 增加彩色强调版本（关键图标用品牌色填充）
- 设计 16px 小尺寸适配版本

**标签**: `#icon` `#line-icons` `#system` `#productivity` `#monochrome`

## Prompt 2: 双色调填充图标

> 双色调风格功能图标，适用于营销和引导页面

**Prompt:**

```text
A duotone icon set of 16 icons for a fintech mobile app, displayed in a 4x4 grid on light gray (#F9FAFB) background, duotone style: primary shapes filled with brand blue (#3B82F6) at full opacity, secondary/background shapes in same blue at 20% opacity creating depth, designed on 48x48px grid, rounded corners (4px radius on shapes), icons include: wallet, credit-card, bank-building, money-transfer, piggy-bank, chart-growth, shield-security, receipt, QR-code, bitcoin, globe-currency, calculator, invoice, fingerprint, bell-alert, pie-chart, each icon self-contained in its space with consistent padding, modern geometric construction, smooth and friendly appearance, suitable for both light and dark backgrounds (shown on light), Stripe/Linear icon quality
```

**示例效果：**

![双色调图标集](images/icon-set-02.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 尺寸 | 1024×1024 | 正方形展示 |
| 风格 | Duotone | 双色调风格 |
| 模型 | GPT-Image-2 | 推荐使用 |

## Prompt 3: iOS App 图标设计

> App Store 应用图标设计，多个候选方案展示

**Prompt:**

```text
A set of 6 iOS app icon design concepts for a meditation/mindfulness app, displayed in 2x3 grid on dark background, each icon is a rounded square (iOS superellipse shape) at 120x120px display size, design explorations: (1) minimal gradient - purple to pink gradient with white lotus silhouette, (2) abstract - concentric circles in calming blue tones creating ripple effect, (3) nature - stylized mountain with crescent moon, dark blue palette, (4) geometric - golden ratio spiral in rose gold on deep navy, (5) typographic - single letter "M" in custom serif with gradient fill, (6) illustrative - side profile face silhouette with flowing hair becoming waves in teal, all following Apple's icon grid system, no text labels on the icons themselves, subtle shadows beneath each icon, professional App Store ready quality, each distinct but cohesive in calming mood
```

**示例效果：**

## Prompt 4: 3D 风格图标

```text
A set of 12 3D rendered icons for a cloud computing platform marketing page, displayed in a 4x3 grid on soft white background, 3D cartoon/clay style with soft shadows and ambient occlusion, pastel color palette: each icon has a distinct pastel base color (mint green, lavender, peach, sky blue, lemon yellow, coral pink), icons include: cloud-server (cloud with rack inside), database-cylinder, shield-lock, rocket-launch, code-brackets, API-connection-nodes, analytics-chart, globe-network, lightning-bolt-speed, puzzle-integration, headset-support, dollar-savings, slight 30-degree rotation for dimensional feel, smooth matte material (clay/plastic look), consistent soft directional lighting from top-left, subtle ground shadow beneath each, Figma 3D / Spline style quality, playful yet professional
```

![3D风格图标集](images/icon-set-04.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
