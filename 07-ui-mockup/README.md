## 💡 通用技巧

1. **设备框架**: `iPhone 15 Pro mockup`、`MacBook screen`、`floating device`
2. **设计风格**: `glassmorphism`（玻璃拟态）、`neumorphism`（新拟物）、`flat design`、`material design`
3. **配色体系**: `dark mode UI`、`vibrant gradient`、`monochrome with accent color`
4. **布局描述**: `card-based layout`、`sidebar navigation`、`tab bar at bottom`
