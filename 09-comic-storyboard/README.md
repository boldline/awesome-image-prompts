---

## 💡 通用技巧

1. **分格布局**: `4-panel comic strip`、`manga page layout with dynamic panels`、`single splash page`
2. **对话气泡**: `speech bubble`、`thought cloud`、`narration box`
3. **运动线条**: `speed lines`、`impact effects`、`motion blur`
4. **情绪表现**: `chibi expression`、`dramatic close-up`、`sweat drop`
