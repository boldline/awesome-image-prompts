## 💡 通用技巧

1. **比例标准**: 演示文稿通常 16:9（1920×1080），使用 `widescreen 16:9 aspect ratio`
2. **留白原则**: `generous white space`、`clean minimal layout`
3. **层次感**: `subtle gradient background`、`frosted glass overlay`、`depth with shadows`
4. **一致性**: 同一 Deck 内保持色调、字体风格、图标风格统一
