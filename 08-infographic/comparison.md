## Prompt 1: 双栏 VS 对比

> 经典的左右对比布局，突出两者差异

**Prompt:**

```text
A bold VS comparison infographic with split-screen layout: left side vs right side comparing "Remote Work vs Office Work". Large "VS" badge in the center dividing the two halves, each side has its own color theme (blue for remote, orange for office), 5-6 comparison points listed vertically with matching icons on each side, visual indicators (progress bars, star ratings, or percentage circles) showing relative strengths, bold headline for each side at the top, key advantages highlighted with checkmark badges, clean modern design with subtle texture background, balanced visual weight on both sides, square format ideal for social media sharing
```

**示例效果：**

![双栏 VS 对比](images/comparison-01.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 尺寸 | 1024×1024 | 方形适合左右平衡构图 |
| 风格 | Bold / Split-screen | 分屏对比风格 |
| 模型 | GPT-Image-2 | 推荐 |

**变体建议：**

- 使用斜线分割代替垂直分割增加动感
- 添加"胜出方"标记（crown/winner badge）
- 改为上下布局适配竖屏阅读

**标签**: `#infographic` `#comparison` `#vs` `#split-screen`

## Prompt 2: 功能矩阵对比表

> 多产品多维度的功能对比表格

**Prompt:**

```text
A feature comparison matrix infographic comparing 3-4 products or plans (Free, Pro, Enterprise) in a clean table layout. Column headers with product/plan names and pricing, rows listing 8-10 features with check marks (green), X marks (red), or partial indicators (yellow dash), the recommended plan column highlighted with a colored border and "Most Popular" badge at top, alternating row backgrounds (white and light gray) for readability, feature category groupings with bold subheaders, icons next to each feature name for quick scanning, professional SaaS-style design with rounded corners and subtle shadows, blue and white primary palette with green accent for CTAs, landscape format suitable for pricing page or sales deck
```

**示例效果：**

![功能矩阵对比表](images/comparison-02.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 尺寸 | 1536×1024 | 横版宽幅适合多列表格 |
| 风格 | SaaS / Table Layout | 产品定价表风格 |
| 模型 | GPT-Image-2 | 推荐 |

**变体建议：**

- 减少为 2 列对比简化信息量
- 使用星级评分代替 check/X 标记
- 添加底部 CTA 按钮区域模拟真实页面

**标签**: `#infographic` `#comparison` `#feature-matrix` `#saas`

## Prompt 3: 信息图式优劣对比

> 用图形化方式展示优缺点分析

**Prompt:**

```text
A pros and cons infographic with creative visual layout comparing advantages and disadvantages of "AI in Education". Two distinct sections: left/top section for pros (green theme) with thumbs-up icon header, right/bottom section for cons (red/coral theme) with thumbs-down icon header. Each point represented as a card with icon + text, arranged in a staggered grid. Visual balance between pros and cons (equal space), a neutral summary/verdict section at the bottom with balanced scale icon, playful illustrated style with rounded shapes and friendly colors, subtle pattern backgrounds differentiating the two sections, clear visual separation with a diagonal or wavy divider line, portrait format for blog or newsletter use
```

**示例效果：**

![信息图式优劣对比](images/comparison-03.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 尺寸 | 1024×1344 | 竖版适合优劣分区展示 |
| 风格 | Playful / Illustrated | 活泼插画风格 |
| 模型 | GPT-Image-2 | 推荐 |

**变体建议：**

- 使用天平/秤的视觉隐喻作为主图
- 改为纯图标阵列减少文字依赖
- 添加权重评分让对比更量化

## Prompt 5: 时间前后对比

> Before/After 变化对比，展示改进效果

**Prompt:**

```text
A before-and-after comparison infographic showing transformation or improvement: "Website Redesign Results". Split into BEFORE (left, desaturated/muted tones) and AFTER (right, vibrant/saturated tones) sections with a vertical divider and sliding indicator arrow. Each side shows the same metrics but with different values: bounce rate, load time, conversion rate, user satisfaction score. Visual representations using circular progress indicators and small bar charts, dramatic improvement shown with upward arrows and percentage change callouts in green, subtle mockup screenshots of the old vs new design in background, professional case-study style layout, gradient from gray (before) to brand blue (after), motivational transformation narrative, landscape format for portfolio or case study presentation
```

**示例效果：**

![时间前后对比](images/comparison-05.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 尺寸 | 1536×1024 | 横版适合前后对照展示 |
| 风格 | Case Study / Professional | 案例研究专业风格 |
| 模型 | GPT-Image-2 | 推荐 |

**变体建议：**

- 使用滑块交互暗示（slider handle）增加趣味
