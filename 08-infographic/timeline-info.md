## Prompt 1: 企业发展历程时间线

> 经典垂直时间线展示公司里程碑，适合关于我们页面

**Prompt:**

```text
A vertical company history timeline infographic showing 8 key milestones from founding to present, central vertical line with alternating left-right milestone cards, each milestone contains: year in large bold typography, event title, brief 1-2 sentence description, and a small representative icon (rocket for launch, handshake for partnership, trophy for award, globe for expansion), year markers as large circles on the central line, connecting lines from circles to cards with subtle curves, corporate professional style with navy blue timeline line, milestone cards as white rounded rectangles with colored left borders (gradient from light blue at founding to deep blue at present), company logo at top, "Our Journey" title in elegant serif font, light gray background with subtle paper texture, suitable for corporate website about page or company brochure
```

**示例效果：**

![企业发展历程时间线](images/timeline-info-01.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 尺寸 | 768×1200 | 竖版适合长时间线展示 |
| 风格 | Corporate Professional | 企业专业风格 |
| 模型 | GPT-Image-2 | 推荐 |

**变体建议：**

- 改为横向时间线适配演示文稿
- 添加配图照片占位区增加真实感
- 使用深色背景配金色时间线打造高端感

**标签**: `#infographic` `#timeline` `#corporate` `#company-history` `#milestones`

## Prompt 2: 产品路线图时间线

> 未来规划路线图，适合产品发布会或投资者沟通

**Prompt:**

```text
A product roadmap timeline infographic showing planned features across 4 quarters, horizontal layout with a road/path metaphor as the timeline base, Q1-Q4 sections clearly divided with distinct background colors (spring green, summer gold, autumn orange, winter blue), each quarter contains 3-4 feature items shown as flag markers or milestone pins on the road, features categorized by type with color-coded tags (new feature in blue, improvement in green, integration in purple), a "you are here" marker showing current progress, completed items shown with checkmarks, upcoming items with outline style, roadmap path illustrated as a winding road with slight 3D perspective, decorative landscape elements (trees, buildings) growing along the path representing growth, modern startup style with playful but professional aesthetic, suitable for product update blog post or stakeholder presentation
```

**示例效果：**

![产品路线图时间线](images/timeline-info-02.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 尺寸 | 1400×600 | 宽幅横版展示季度规划 |
| 风格 | Startup Playful / Road Metaphor | 创业公司活泼风格 |
| 模型 | GPT-Image-2 | 推荐 |

**变体建议：**

- 改为甘特图样式展示时间跨度和并行项目
- 使用火箭发射轨迹作为视觉隐喻
- 简化为纯文字+图标的极简路线图

**标签**: `#infographic` `#timeline` `#roadmap` `#product` `#startup`

## Prompt 3: 历史事件时间线

> 教育风格的历史事件时间线，适合教学或科普内容

**Prompt:**

```text
A historical events timeline infographic showing the evolution of the internet from 1960s to 2020s, horizontal flowing timeline with era-based sections, each era styled differently to reflect its period (retro pixelated for 80s, web 1.0 style for 90s, glossy for 2000s, flat design for 2010s, modern for 2020s), key events marked with illustrated vignettes: ARPANET creation, first email, World Wide Web invention, dot-com boom, social media rise, mobile era, AI revolution, event descriptions in vintage typewriter font transitioning to modern sans-serif, color palette evolving from monochrome green (terminal era) to full vibrant spectrum (modern era), decorative elements reflecting each period (floppy disks, dial-up modems, smartphones), educational poster style suitable for classroom wall or museum exhibit, includes date markers and connecting flow lines
```

## Prompt 4: 项目甘特图时间线

> 项目管理甘特图风格，适合团队协作和进度汇报

**Prompt:**

```text
A Gantt chart style project timeline infographic showing a 6-month project plan, horizontal bar chart layout with time axis (months) across the top, task categories listed vertically on left side (Research, Design, Development, Testing, Launch), each task shown as a colored horizontal bar spanning its duration, overlapping bars showing parallel workstreams, milestone diamonds at key delivery dates, dependency arrows connecting related tasks, current date indicated by vertical red line, completed portions filled solid vs remaining portions with diagonal stripes, resource allocation indicators (team member avatars or initials) beside each bar, clean project management tool aesthetic with white background, task bars in coordinated colors (blue spectrum for development, green for testing, orange for design), grid lines for weekly intervals, progress percentage labels on each bar, suitable for project status report or PMO dashboard
```

**参数说明：**

| 参数 | 推荐值 | 说明 |

| 风格 | Project Management / Clean Data | 项目管理工具风格 |
| 模型 | GPT-Image-2 | 推荐 |
