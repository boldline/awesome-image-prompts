## 💡 通用技巧

1. **布局模式**: `single-column`（单列）、`grid layout`（网格）、`modular sections`（模块化）
2. **图标风格**: `line icons`、`filled icons`、`isometric icons`、`hand-drawn icons`
3. **色彩方案**: 通常使用 3-5 色体系，主色+辅色+强调色
4. **数据层次**: 用字号大小和颜色深浅区分数据重要性
