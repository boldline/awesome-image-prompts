## Prompt 1: 用户注册引导流程

> 线性步骤流程图，适用于产品使用指南或文档配图

**Prompt:**

```text
An informative process flow infographic showing user onboarding steps in 5 stages, clean horizontal layout with connected steps flowing left to right, each step presented as a rounded rectangle card containing: step number in a colored circle, descriptive icon (signup form, email verification, profile setup, tutorial, dashboard), short title and one-line description below, connecting arrows between steps with subtle animation feel (dotted or gradient lines), progress bar running along the top showing completion percentage, modern SaaS product style with soft blue and purple gradient accents, white background with light geometric patterns, friendly sans-serif typography, suitable for help documentation or welcome email
```

**示例效果：**

![用户注册引导流程](images/process-flow-01.png)

**参数说明：**

| 参数 | 推荐值 | 说明 |
|------|--------|------|
| 尺寸 | 1200×600 | 横版适合网页嵌入 |
| 风格 | Modern SaaS / Flat | 现代互联网产品风格 |
| 模型 | GPT-Image-2 | 推荐 |

**变体建议：**

- 改为竖版瀑布流布局适配移动端展示
- 添加分支路径展示不同用户类型的注册差异
- 使用手绘风格线条和图标增加亲和力
