## 💡 通用技巧

1. **视觉风格选择**:
   - `dark theme with neon accents` - 暗色主题，适合技术博客
   - `clean whiteboard style` - 白板风格，适合教学
   - `isometric 3D` - 等轴测，适合架构展示
   - `hand-drawn sketch` - 手绘风，适合头脑风暴
   - `corporate professional` - 企业风，适合正式文档
   - `blueprint style` - 蓝图风，适合工程图纸
   - `gradient modern` - 渐变现代风，适合PPT
2. **连接线风格**: `straight connectors`、`curved bezier paths`、`orthogonal routing`
3. **图标系统**: `AWS/Azure/GCP official icons`、`generic tech icons`、`minimal geometric shapes`
4. **标注方式**: `numbered annotations`、`color-coded legend`、`inline labels`
